# 词法分析

词法分析：是从句子中切分出单词，找出词汇的各个词素，从中获得单词的语言学信息并确定单词的词义。

不同的语音对词法分析有不同的要求：

- 在英语等语言中，音位单词之间是以空格自然分开的，切分一个单词很容易，所以找出句子的各个词汇就很方便。
  - 但是，由于英语单词有词性、数、时态、派生及变形等变化，要找出各个词素就复杂得多，需要对词尾或词头进行分析。
  - 通常，词法分析可以从词素中获得许多有用的语言学信息，这些信息对于句法分析是非常有用的。
    - 如英语中构成词尾的词素s通常表示名词复数或动词第三人称单数。
    - 自然语言理解系统中的电子词典一般只放词根，并支持词素分析，这样可以大大压缩电子词典的规模。
      - 一个词可以有许多的派生、变形，如work，可以变化出works，worked，working，worker，workable等。如果将这些派生的、变形的词全放入词典、将会产生非常庞大的数据量。实际上它们的词根只有一个。
- 在汉语中，每个字就是一个词素，所以要找出各个词素是相当容易的。
  - 但要切分出各个词就非常困难，不仅需要构词的知识，还需要解决可能遇到的切分歧义。

下边是一个英语词法分析的算法，可以对那些按英语文法规则变化的英语单词进行分析：

```
repeat
	look for word in dictionary
	if not found
	then modify the word
Until word is found or no further modification possible
```

其中，word是一个变量，初始值是当前的单词。

例如，对于单词catches、ladies可以进行如下分析：

catches	ladies	词典中查不到

catche	  ladie	  修改1：去掉s

catch  	  ladi         修改2：去掉e

​				lady		修改3：把i变成y

英语词法分析的难度在于词义判断，因为单词往往有多种解释，仅仅依靠词典常常无法判断。要判定单词的词义只能依靠句子中其他相关单词和词组的分析。
